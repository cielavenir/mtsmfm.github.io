例えば1分ごとに何かしたいときとか\
cron使うの面倒なときに\
\
application\_controller.rb

~~~~ {.syntax-highlight}
before_filter :work_something
def work_something
  unless Rails.cache.read(:cron_flag)
    #なんかする
    Rails.cache.write(:cron_flag, true, expires_in: 60)
  end
end
~~~~

ものすごくアクセスが多いと無駄が多すぎるだろうけど。
